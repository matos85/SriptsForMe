# Jupyter Notebook Launcher

## Описание
Этот проект предоставляет простой способ запуска Jupyter Notebook с автоматической проверкой его установки.
Если Jupyter не установлен, скрипт автоматически установит его перед запуском. После запуска Jupyter Notebook, он будет автоматически открыт в вашем браузере. 
Если браузер закрыть, терминал автоматически закроется.

## Содержимое проекта
- `JNI.py`: Python-скрипт, который запускает Jupyter Notebook.
- `Jupyter Notebook.bat`: BAT файл для запуска Python-скрипта в командной строке.

## Установка

### Требования
- Python версии 3.x установлен на вашем компьютере. 
- Убедитесь, что `pip` установлен и доступен из командной строки.
- Система должна поддерживать запуск Jupyter Notebook (Windows, macOS, Linux).

### Шаги по установке

1. **Скачать проект**
   - Клонируйте репозиторий или скачайте файлы `JNI.py` и ``Jupyter Notebook.bat` в желаемую директорию на вашем компьютере.

2. **Установка Python**
   - Если Python ещё не установлен,  установите его из архива. Во время установки выберите опцию "Add Python to PATH".
или
   - **Добавление Python в PATH вручную**:
   1. Найдите, где установлен Python. Обычно это что-то вроде `C:\Python39` или `C:\Users\<ВашеИмяПользователя>\AppData\Local\Programs\Python\Python312`.
   2. Скопируйте путь к папке, в которой установлен Python и содержится файл `python.exe`.
   3. Откройте "Настройки системы" (поиск "Настройки" в меню Пуск).
   4. Перейдите в раздел "Система" → "О системе" и выберите "Дополнительные параметры системы".
   5. Нажмите на кнопку "Переменные среды".
   6. В разделе "Системные переменные" найдите переменную `Path` и выделите её, затем нажмите "Изменить".
   7. В открывшемся окне нажмите "Создать" и вставьте скопированный ранее путь к папке с Python. Также добавьте путь к папке `Scripts`, например, `C:\Users\<ВашеИмяПользователя>\AppData\Local\Programs\Python\Python312\Scripts`.
   8. Нажмите "ОК", чтобы сохранить изменения, и закройте все окна.

3. **Проверка установки Python**
   - Откройте командную строку (нажмите `Win + R`, введите `cmd` и нажмите `Enter`).
   - Введите команду:
     ```bash
     python --version
     ```
   - Убедитесь, что версия Python отображается правильно.

4. **Установка Jupyter Notebook**
   - Для установки Jupyter Notebook выполните следующую команду в командной строке:
     ```bash
     pip install jupyter
     ```

## Использование

1. **Запуск проекта**
   - Дважды щёлкните на файл ` Jupyter Notebook.bat`.
   - Скрипт проверит, установлен ли Jupyter Notebook.
   - Если Jupyter не установлен, он будет автоматически установлен.
   - После этого Jupyter Notebook запустится, и он откроется в браузере.

2. **Закрытие Jupyter Notebook**
   - Закройте браузер, чтобы завершить работу Jupyter Notebook.
   - После закрытия браузера, командное окно тоже закроется автоматически.

## Устранение неполадок
- **Ошибка "python не является внутренней или внешней командой"**:
  - Убедитесь, что Python установлен и его путь добавлен в переменную окружения PATH.

- **Jupyter не устанавливается**:
  - Убедитесь, что pip установлен и доступен из командной строки. Для обновления pip выполните:
    ```bash
    python -m pip install --upgrade pip
    ```

- **Проблемы с запуском Jupyter Notebook**:
  - Убедитесь, что все зависимости установлены, и перезапустите командную строку или терминал.
  
- **Ошибка, связанная с портом**:
  - Если Jupyter не может запуститься из-за использования порта (например, 8888), попробуйте запустить его на другом порту с помощью команды:
    ```bash
    jupyter notebook --port=8889
    ```
